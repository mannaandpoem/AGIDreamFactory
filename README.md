
# AGI梦工厂

**构建更智能的世界**

欢迎来到AGI梦工厂，这是一个致力于通过实践实施和理论讨论来探索人工通用智能（AGI）的各个方面的协作项目。

## 项目结构

该项目被组织成几个子目录，每个目录都专注于AGI研究和应用的不同领域：

### [大语言模型宇宙](./llm_universe)
大语言模型宇宙目录致力于大型语言模型（LLM），提供架构配置、比较和深入分析。主要资源包括：
- [大模型架构配置概览](./llm_universe/llm_architecture_configuration.md)
![大模型架构配置](./llm_universe/llm_architecture_configuration.jpeg)
- [全球热门大模型架构配置信息汇总](./llm_universe/国内外30个热门大模型架构配置信息汇总.pdf)

### [智能体联盟](./agent_league)
智能体联盟目录包含与智能体的开发和挑战相关的资源和讨论。有关更多详细信息，请参阅此目录中的[README](./agent_league/README.md)文件。

### [AI哲学](./ai_philosophy)
该部分深入探讨了人工智能的哲学问题和影响。要进行更深入的讨论，请参阅AI哲学目录中的[README](./ai_philosophy/README.md)文件。

### [AIGC艺术](./aigc_art)
AIGC艺术专注于AI生成内容及其对艺术行业的影响。探索这个目录的[README](./aigc_art/README.md)，了解我们正在进行的项目和实验。
